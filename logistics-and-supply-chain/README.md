## Usage

Reports are automatically generated by the VeriCrop system and exported to this directory. The system maintains:
- Real-time export of all batch and supply chain events
- Aggregated metrics computed from historical data
- Timeline logs with timestamps for traceability

## Data Retention

Generated files are retained according to regulatory requirements:
- Batch data: Indefinite retention
- Supply chain events: Minimum 7 years
- Reports: As needed for auditing

## Integration

This directory integrates with:
- **Kafka**: Real-time event streaming for supply chain events
- **Airflow**: Workflow orchestration for batch processing
- **PostgreSQL**: Metadata storage and querying
- **Blockchain**: Immutable audit trail
